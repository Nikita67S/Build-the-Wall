# Калькулятор строительных материалов для стены

Модульное приложение на Python для автоматического расчета точного количества кирпича и массы строительного раствора, необходимых для возведения стены по заданным размерам.

## 📂 Структура проекта

Проект организован по принципу разделения обязанностей (Separation of Concerns). Все расчетные модули сгруппированы в логические пакеты внутри директории `buildWall`:

```text
buildWall/
├── CalcBricks/              # Модули для расчета кирпича
│   ├── count_of_bricks.py     # Расчет чистого количества кирпичей в штуках
│   └── percent_bricks_in_wall.py # Вычисление объема, занимаемого кирпичом
├── CalcCommon/              # Общие базовые модули и ресурсы
│   ├── constants.py           # Физические константы, нормы расхода и резервы
│   ├── Stock.py               # Расчет финального запаса материалов (склад)
│   └── volume_wall.py         # Расчет общего геометрического объема стены
└── CalcMortar/              # Модули для расчета раствора
    ├── volume_mortar_in_kg.py # Конвертация объема раствора в массу (кг)
    └── volume_mortar_in_wall.py # Нахождение объема раствора в кубических метрах
```

Главная точка входа в программу — файл **`app.py`**, расположенный в корне папки `buildWall`. Он управляет всей цепочкой вызовов и связывает модули через передачу аргументов в функции.

## 🛠️ Требования и запуск

Для работы скрипта необходим установленный интерпретатор **Python 3.x**.

### Инструкция по запуску:
1. Откройте терминал в корневой папке проекта (`PythonProject2`).
2. Запустите главный модуль, выполнив команду:
   ```bash
   python buildWall/app.py
   ```

## 💻 Использование и настройка

### Изменение размеров стены:
Чтобы рассчитать материалы для другой стены, откройте файл `buildWall/app.py` и отредактируйте кортеж `wall_lwh` (все значения передаются в **миллиметрах**):

```python
# Размеры стены: длина, ширина, высота в мм
wall_lwh = (10000, 510, 3000)
```

### Изменение строительных нормативов:
Если вам нужно изменить коэффициенты запаса, норму расхода раствора или процентное заполнение элементов в кладке, отредактируйте соответствующие переменные в файле `buildWall/CalcCommon/constants.py`.

## 📋 Пример вывода программы
При успешном запуске приложение выведет в консоль итоговые результаты снабжения:
```text
Всего нужно столько кирпичей: [Вычисленное_количество_штук]
Всего нужно столько раствора в кг: [Вычисленная_масса_в_кг]
```
