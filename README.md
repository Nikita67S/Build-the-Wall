# Калькулятор строительных материалов для стены

Скрипт на Python для расчета точного количества кирпича и массы раствора, необходимых для постройки стены по заданным размерам с учетом коэффициентов запаса.

## 🚀 Архитектура проекта

Проект реализован по модульному принципу. Каждый этап расчетов вынесен в отдельный файл (библиотеку функций):

*   `app.py` — Главная точка входа. Управляет цепочкой вызовов и выводит результат.
*   `volume_wall.py` — Переводит размеры из мм в метры и считает общий объем стены.
*   `percent_bricks_in_wall.py` — Вычисляет долю объема, занимаемую кирпичом.
*   `count_of_bricks.py` — Рассчитывает чистое количество кирпичей в штуках.
*   `volume_mortar_in_wall.py` — Находит объем раствора в кубических метрах.
*   `volume_mortar_in_kg.py` — Конвертирует объем раствора в общую массу (кг).
*   `Stock.py` — Применяет коэффициенты резерва (запаса) на бой и усадку.
*   `constants.py` — Хранит физические константы и параметры материалов.

## 🛠️ Требования и установка

Для запуска проекта необходим установленный интерпретатор **Python 3.x**.

1. Клонируйте репозиторий или скачайте все файлы в одну папку.
2. Убедитесь, что все модули находятся в одной директории.

## 💻 Использование

Чтобы изменить размеры возводимой стены, откройте файл `app.py` и отредактируйте переменную `wall_lwh` (значения указываются в **миллиметрах**):

```python
# Длина, Ширина, Высота стены в мм
wall_lwh = (10000, 510, 3000)
```

Для изменения строительных нормативов (расход, процент заполнения, запас) отредактируйте параметры в `constants.py`.

### Запуск проекта:
Выполните команду в терминале:
```bash
python app.py
```
